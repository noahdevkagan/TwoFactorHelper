2FA Helper

Auto-copies verification codes from iMessage to your clipboard.
No more switching apps to copy a 6-digit code.

Download for macOS

macOS 13+ · Apple Silicon & Intel · Free & open source

⚡

Instant Detection

Monitors iMessage every 2 seconds. Code is on your clipboard before you switch apps.

🔔

Native Notifications

Get a macOS notification showing the code when it arrives. Also shown in the menu bar.

🛡

Privacy First

Runs 100% locally. No network calls, no analytics, no data leaves your Mac.

🧰

Smart Parsing

Recognizes codes from Google, Uber, banks, and most 2FA formats automatically.

Setup

  1. Download and unzip TwoFactorHelper.zip
  2. Move TwoFactorHelper.app to your Applications folder
  3. Important: Right-click the app and choose Open the first time (macOS blocks unsigned apps). If you see "damaged" warning, open Terminal and run: xattr -cr /Applications/TwoFactorHelper.app
  4. Open System Settings > Privacy & Security > Full Disk Access and add TwoFactorHelper
  5. Launch the app — look for the shield icon in your menu bar
  6. That's it! First launch may take a moment to install dependencies. Next time you get a 2FA text, the code auto-copies to your clipboard
